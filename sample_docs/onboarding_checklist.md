# 台灣科技集團股份有限公司 — 新進員工入職清單

> 人力資源部 | 版本：v3.0 | 2025年更新
> 此文件供新進員工與其直屬主管共同使用，記錄入職過程中的所有待辦事項。

---

## 1. 到職前準備（HR / 主管執行）

**到職前2週：**
- [ ] HR 寄送錄取通知與到職須知郵件
- [ ] 主管完成職位設定（系統權限、座位安排）
- [ ] IT 部門建立帳號（Email、Slack、Jira等）
- [ ] 安排 Buddy（指導員），傳送歡迎訊息

**到職前3天：**
- [ ] IT 部門準備好設備（筆記型電腦、周邊）
- [ ] HR 發送報到地點與流程說明
- [ ] 主管在 Slack 頻道發布歡迎訊息

## 3. 第一週（Day 2-5）

### 產品與系統了解

- [ ] 觀看台灣科技產品介紹影片（共3支，約2小時）
- [ ] 參加產品 Demo 示範（由產品部安排，每週三14:00）
- [ ] 閱讀系統架構文件（技術人員必讀，Confluence: 技術/架構）
- [ ] 取得測試環境存取權限

### 流程與規範

- [ ] 閱讀工程最佳實踐手冊（工程師）/ 銷售SOP（業務）
- [ ] 完成資安意識訓練模組二（釣魚辨識）
- [ ] 了解費用報銷流程（HR System → 費用申請）
- [ ] 了解請假流程（HR System → 假勤申請）

### 人際融入

- [ ] 與 Buddy 進行第一次正式 Coffee Chat
- [ ] 參加部門週會（了解當前工作重點）
- [ ] 與跨部門夥伴進行至少3次 Coffee Chat（HR安排名單）
- [ ] 加入公司社團（運動、攝影、讀書會等，看公告）

## 4. 第一個月（Week 2-4）

### 工作融入

- [ ] 接手第一個任務（與主管確認範圍）
- [ ] 完成任務後進行首次成果分享（5分鐘）
- [ ] 參與至少一次產品企劃或技術設計討論

### 學習計畫

- [ ] 完成線上訓練課程「台灣科技集團股份有限公司 核心產品認證」（約8小時）
- [ ] 閱讀過去6個月的關鍵決策文件（ADR / 產品PRD）
- [ ] 與各部門主管進行 Stakeholder Meeting（HR安排）

### 月底確認

- [ ] 與主管進行第一個月回顧（1:1）
- [ ] 填寫《新人入職體驗問卷》（HR寄送）
- [ ] 確認30天目標的完成狀況，設定60天目標

## 5. 三個月目標（Month 2-3）

### 能力發展目標

| 月份 | 學習重點 | 評估方式 |
|------|---------|---------|
| 第2個月 | 深入了解負責領域，獨立完成任務 | 主管評分 |
| 第3個月 | 開始分擔部分導師/資深同仁職責 | 360度回饋 |

### 試用期末評估（通常為3個月）

評估維度：
1. 工作成果（40%）— 任務完成質量與效率
2. 學習能力（30%）— 吸收新知識的速度與主動性
3. 團隊合作（20%）— 溝通協作與文化契合
4. 職業態度（10%）— 出勤、責任感、積極性

試用期結果：正式確認轉正 / 延長試用期（最多3個月）/ 終止合約

## 7. 常見問題 FAQ

**Q：薪資什麼時候入帳？**
A：每月25日，若遇假日則提前至前一個工作日。

**Q：忘記密碼怎麼辦？**
A：至 https://sso.twtech.com.tw 點選「忘記密碼」，或聯絡 IT 服務台。

**Q：加班費如何申請？**
A：至HR System填寫加班申請，主管審核後自動計入薪資。月薪制員工依勞基法規定計算加班費。

**Q：特休假怎麼計算？**
A：到職日起6個月後享有3天，滿1年享有7天，依勞基法遞增。可在HR System查詢餘額。

**Q：想申請出差怎麼辦？**
A：填寫出差申請單（HR System），主管簽核後可訂機票/飯店，費用憑收據報銷。

聯絡HR：hr@twtech.com.tw | 分機 302-304
