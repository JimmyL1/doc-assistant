## 1. 帳號管理

**Q1：如何建立組織帳號？**

A：至 https://app.twtech.com.tw/signup 填寫組織資訊，選擇方案後以信用卡或電匯付款即可完成建立。企業方案（50人以上）建議聯絡業務顧問：sales@twtech.com.tw。

**Q2：如何邀請團隊成員加入？**

A：管理員登入後至「設定 → 成員管理 → 邀請成員」，輸入Email即可發送邀請信。受邀者須在72小時內點擊連結完成設定，過期需重新邀請。

**Q3：忘記密碼如何重設？**

A：在登入頁面點擊「忘記密碼」，輸入帳號Email後系統會發送重設連結（有效期15分鐘）。若未收到郵件，請確認垃圾信件夾或聯絡客服。

**Q4：如何設定多因素驗證（MFA）？**

A：至「設定 → 安全性 → 多因素驗證」，選擇 Google Authenticator 或 SMS 驗證。企業方案管理員可強制所有成員啟用 MFA。強烈建議開啟以保護帳號安全。

**Q5：員工離職後如何處理帳號？**

A：管理員至「設定 → 成員管理」找到該成員，點選「停用帳號」。停用帳號不會刪除其上傳的文件，資料仍保留在組織帳號中。如需完全移除，請選擇「刪除成員」。

**Q6：可以有多個管理員嗎？**

A：可以。至「設定 → 成員管理」，對成員點選「更改角色 → 管理員」即可。建議至少設定2名管理員以防單點失效。

## 2. 計費問題

**Q7：有哪些付費方案？**

A：台灣科技提供三種方案：
- **Starter**：NT$2,990/月，5名使用者，50GB儲存，1,000次AI查詢
- **Business**：NT$9,990/月，25名使用者，500GB儲存，10,000次AI查詢
- **Enterprise**：客製化定價，無限使用者，自訂儲存，無限查詢

年付方案享85折優惠。詳細比較請參考：https://www.twtech.com.tw/pricing

**Q8：如何升級或降級方案？**

A：至「設定 → 訂閱方案 → 更改方案」。升級立即生效，系統按比例計算差額。降級在下個計費週期生效，期間仍享原方案權益。

**Q9：AI查詢次數超過怎麼辦？**

A：超過後系統會自動暫停AI查詢功能，您的文件資料不受影響。可選擇：(1) 購買額外查詢包（NT$500/1,000次），(2) 升級至更高方案，(3) 等待下個計費週期重置。

**Q10：支援哪些付款方式？**

A：接受信用卡（Visa/MasterCard/JCB）、銀行轉帳、電子發票。政府機關與大型企業可申請月結付款（需審核）。如需統一編號開立發票，請在結帳時填寫。

**Q11：可以申請退款嗎？**

A：年付方案在購買後14天內可申請全額退款（需符合試用條款）。月付方案不提供退款，但可取消訂閱，於當期結束前仍可使用。

**Q12：有提供非營利組織或學術機構優惠嗎？**

A：有的。符合資格的非營利組織享有Business方案六折優惠，學術機構享有七折優惠。請發送組織相關證明文件至 ngo@twtech.com.tw 申請審核。

## 3. 功能操作

**Q13：支援哪些文件格式？**

A：目前支援以下格式：
- 文件：PDF、Word（.docx）、PowerPoint（.pptx）、TXT、Markdown
- 試算表：Excel（.xlsx）—— 限資料型工作表，不支援圖表
- 圖片中的文字（OCR）：JPG、PNG、TIFF（準確率依圖片清晰度而定）

單一文件上限：100MB。每次可批量上傳最多20個文件。

**Q14：文件上傳後多久可以查詢？**

A：一般PDF/DOCX文件在1-3分鐘內完成處理（取決於文件大小與複雜度）。大型文件（>50頁）可能需要5-10分鐘。系統會在處理完成後發送通知。

**Q15：如何讓AI只在特定文件中尋找答案？**

A：在查詢時點選「文件篩選」，勾選您想查詢的文件或資料夾。若不指定，系統預設搜尋您有權限存取的所有文件。

**Q16：AI的回答準確嗎？如何提高準確度？**

A：準確度取決於文件品質和問題的具體性。提升準確度的建議：
- 問題越具體越好（避免過於模糊的問題）
- 確保上傳的文件是可搜尋的文字（非掃描圖片）
- 提供上下文（例如：「根據2025年Q1財報，...」）
- 對AI回答給予正負回饋，協助系統持續優化

**Q17：可以匯出查詢結果嗎？**

A：可以。在查詢結果頁面點選「匯出」，支援格式：PDF報告、Word文件、JSON（含來源引用）。Enterprise方案支援批量匯出與 API 自動化匯出。

## 4. 整合問題

**Q18：台灣科技平台可以和哪些系統整合？**

A：官方支援的整合包括：
- 生產力工具：Microsoft 365、Google Workspace、Slack
- 開發工具：GitHub、GitLab、Jira
- 儲存服務：Google Drive、OneDrive、SharePoint、Dropbox
- ERP：SAP、Oracle ERP Cloud、Microsoft Dynamics
- 自訂整合：透過 REST API 或 Webhook

完整整合清單：https://docs.twtech.com.tw/integrations

**Q19：如何設定 Microsoft 365 整合？**

A：至「設定 → 整合 → Microsoft 365」，點選「連接」並以 Microsoft 帳號授權。授權後可選擇同步特定 SharePoint 網站或 OneDrive 資料夾。注意：需要 Microsoft 365 全域管理員授權。

**Q20：API 有速率限制嗎？**

A：是的，各方案的限制如下：
- Starter：60 requests/分鐘
- Business：300 requests/分鐘
- Enterprise：自訂（預設1,000 requests/分鐘）

超過限制會收到 HTTP 429 錯誤，回應 Header 的 `Retry-After` 會告知等待時間。

## 5. 技術問題

**Q21：資料儲存在哪裡？是否符合台灣法規？**

A：所有資料預設儲存於台灣本地資料中心（中華電信 IDC，台北/桃園雙機房）。Enterprise方案可選擇私有雲部署或指定資料中心。台灣科技集團股份有限公司符合個資法、ISO 27001:2022 及 SOC 2 Type II 標準。

**Q22：資料如何加密？**

A：
- 傳輸加密：TLS 1.3
- 靜態加密：AES-256
- 向量嵌入加密：自訂加密索引
- 金鑰管理：AWS KMS（Enterprise可使用自帶金鑰 BYOK）

**Q23：如果我的帳號被盜用怎麼辦？**

A：請立即執行以下步驟：
1. 若仍可登入，立即更改密碼並啟用MFA
2. 若無法登入，聯絡客服電話 (02) 2345-6789 要求緊急鎖定帳號
3. 審查最近的存取紀錄（設定 → 安全性 → 存取紀錄）
4. 通知您的管理員

**Q24：平台的SLA保證是多少？**

A：
- Starter / Business：99.9% 可用性（每月允許停機 ~43分鐘）
- Enterprise：99.95% 可用性（每月允許停機 ~22分鐘）

SLA計算排除預先公告的維護時間（每月第一個週日 02:00-06:00）。超過SLA可申請服務積分。即時狀態：https://status.twtech.com.tw

**Q25：如何匯出我的所有資料（資料可攜性）？**

A：至「設定 → 資料管理 → 匯出所有資料」。系統將打包您的所有文件、查詢歷史與設定，以加密ZIP格式提供下載（最多保留72小時）。Enterprise方案支援自動定期備份至指定S3/Azure Blob。
